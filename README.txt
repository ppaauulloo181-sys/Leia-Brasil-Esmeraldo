Biblioteca Escolar — Sistema estático de empréstimos

Como usar:
1. Baixe o ZIP e extraia.
2. Abra index.html no navegador.
3. Use o menu 'Admin' para adicionar livros ou recriar a amostra 710.
4. Registre empréstimos pela página de detalhes do livro; devolva pela aba Empréstimos.
5. Todos os dados ficam em localStorage do navegador (pode exportar CSV dos empréstimos).

Para produção: crie backend (Node/Express + PostgreSQL) para persistência, autenticação e integrações.
